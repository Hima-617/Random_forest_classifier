# Object Detection & Human Pose Estimation

## Overview

This project detects objects and estimates human body poses using Python, OpenCV, and MediaPipe.

## Features

- Object Detection
- Human Pose Estimation
- Real-time Webcam Detection
- Body Landmark Tracking

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## How to Run

1. Install the required libraries:

```bash
pip install opencv-python mediapipe numpy
```

2. Open the Jupyter Notebook:

```bash
jupyter notebook
```

3. Run the notebook:

```text
Object_detection_&_humanpose_estimation.ipynb
```

## Applications

- Fitness Tracking
- Sports Analysis
- Human Activity Recognition
- Gesture Detection
